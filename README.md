Jetpack Compose Template
=========

![Android CI](https://github.com/whyrising/Jetpack-Compose-Template/workflows/Android%20CI/badge.svg)

*Up-To-Date Jetpack Compose project template with Kotlin for source code and Gradle build scripts.*

#### Running on:

* Android Studio Arctic Fox | 2020.3.1 Beta 4

#### Dependencies:

* Jetpack Compose
* [Kotest](https://kotest.io/) framework for unit tests
* [y](https://github.com/whyrising/y) library for collections
